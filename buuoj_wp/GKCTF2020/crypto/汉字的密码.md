# 汉字笔画出头数对照表
hanzi_stroke_dict = {
    '田': 0, '口': 1, '由': 2, '人': 2, '中': 3,
    '工': 4, '大': 5, '土': 5, '士': 5, '王': 6,
    '夫': 7, '井': 8, '羊': 9, '壮': 9
}

def decrypt_dangpu_cipher(cipher_text):
    result = ""
    # 移除空格并遍历每个字符
    for char in cipher_text.replace(" ", ""):
        if char in hanzi_stroke_dict:
            result += str(hanzi_stroke_dict[char])
        else:
            result += f"[{char}]"
    return result

# 使用示例
cipher = "王壮 夫工 王中 王夫 由由井 井人 夫中 夫夫 井王 土土 夫由 土夫 井中 士夫 王工 王人 土由 由口夫"
print(decrypt_dangpu_cipher(cipher))

## 如何识别当铺密码（汉字笔画出头数密码）

### 1. 题目特征分析

在CTF比赛中，当铺密码通常具有以下特征：

- **全部由汉字组成**：题目中给出的所有字符都是汉字，没有阿拉伯数字或英文字母
- **重复字符较多**：如"由由井"、"夫夫"、"土土"等，相同汉字重复出现
- **字符种类有限**：使用的汉字种类相对较少，且多为结构简单的常用字
- **分组排列**：字符按一定规律分组，如"王壮 夫工 王中..."

### 2. 识别思路

#### 方法一：搜索匹配
当遇到这种题目时，可以：
1. 将题目中的字符组合在搜索引擎中搜索
2. 查找是否有类似的CTF题目或WP（Write-Up）
3. 通过搜索结果确定密码类型

#### 方法二：特征分析
观察字符特点：
- 都是简单汉字，笔画数不多
- 有很多结构对称或有规律的字（如"王"、"由"、"田"、"口"等）
- 相同字符重复出现，暗示可能有固定的映射关系

### 3. 为什么是"笔画出头数"？

#### 汉字结构分析
让我们分析一下这些汉字的特点：

1. **"王"字**：有6个笔画"出头"（上、下、左、右各一，中间两横各一）
2. **"由"字**：有2个笔画"出头"（上、下各一）
3. **"田"字**：有0个笔画"出头"（四边封闭）
4. **"口"字**：有1个笔画"出头"（通常指开口方向）
5. **"井"字**：有8个笔画"出头"（四个方向的延伸）

这种规律性很强的特征暗示我们可能存在一种固定的映射关系。

### 4. 实际识别过程

在真实的CTF比赛中，选手可能会这样识别：

1. **初步观察**：全是汉字，有重复，猜测是某种编码
2. **尝试常见方法**：
   - 拼音首字母：王(W)壮(Z)→WZ？但后续难以处理
   - 笔画数：王(4画)壮(6画)→46？但不符合数字规律
3. **深入分析结构**：
   - 发现"王"、"由"、"田"、"口"等字具有明显的几何特征
   - 尝试计算"出头"笔画数，发现能对应到0-9数字
4. **验证假设**：
   - 查找"当铺密码"相关资料确认
   - 用部分字符验证规律是否成立

### 5. 为什么叫"当铺密码"？

这个名称来源于中国古代当铺的记账方式：
- 当铺为了防止账目被篡改，会使用特殊的符号或汉字记录数字
- 这些汉字的"出头"笔画数正好对应0-9的数字
- 因此得名"当铺密码"

### 6. 解题技巧总结

对于这类题目，建议采用以下识别流程：

1. **观察字符特征**：

2. **尝试常见编码**：

   - 结构特征分析

3. **查找规律**：
   - 相同字符是否对应相同数字
   - 是否能建立一一对应的映射关系

4. **验证假设**：
   - 搜索相关WP或资料
   - 用已知规律解释所有字符

5. **得出结论**：

# CTF解题大全

这是一个全面的CTF（Capture The Flag）竞赛解题指南，涵盖了从基础到高级的各种题型和解题技巧。

## 项目结构

```
CTF_解题大全/
├── 01_基础与思维方法论.md
├── 02_Web安全题型详解.md
├── 03_PWN题型详解.md
├── 04_Reverse_Engineering题型详解.md
├── 05_密码学题型详解.md
├── 06_Miscellaneous题型详解.md
├── 07_AI_Security题型详解.md
└── 目录.md
```

## 内容概览

本指南详细介绍了以下CTF题型：

1. **基础与思维方法论** - CTF竞赛入门知识和解题思路
2. **Web安全** - SQL注入、XSS、文件上传等Web漏洞利用
3. **PWN** - 二进制漏洞利用技术
4. **逆向工程** - 程序逆向分析方法
5. **密码学** - 各种加密算法和破解技术
6. **Miscellaneous** - 隐写术、取证、编码等综合技能
7. **AI Security** - 人工智能安全相关的新兴题型

## 特色内容

- **从零开始**：适合初学者的详细教程
- **实战案例**：丰富的实际题目解析和EXP代码
- **工具介绍**：各类CTF工具的使用方法
- **AI安全**：涵盖最新的AI安全CTF题型
- **完整EXP**：提供可直接运行的漏洞利用代码

## 使用方法

1. 从[目录.md](目录.md)开始浏览
2. 按章节顺序学习或选择感兴趣的主题
3. 结合实际CTF平台进行练习
4. 参考EXP代码理解漏洞利用原理

- CTF初学者

- 想要提升CTF技能的参赛者
